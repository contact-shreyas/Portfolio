## 🔄 How to Update Your Portfolio

### Option 1: Make Changes Locally & Redeploy

1. Edit files in `d:\1. Portfolio\app\page.tsx`
2. Test locally: `npm run dev`
3. Deploy changes: `vercel --prod`

### Option 2: Connect to GitHub (Recommended)

This enables automatic deployments on every git push:

1. **Create GitHub repository:**
   - Go to https://github.com/new
   - Name it: `portfolio` or `shreyas-portfolio`
   - Keep it public or private
   - Don't initialize with README

2. **Push your code:**
   ```powershell
   cd "d:\1. Portfolio"
   git remote add origin https://github.com/contact-shreyas/YOUR-REPO-NAME.git
   git branch -M main
   git push -u origin main
   ```

3. **Connect to Vercel:**
   - Go to https://vercel.com/dashboard
   - Click your project "shreyas-gupta-portfolio"
   - Go to Settings → Git
   - Connect your GitHub repository

4. **Auto-deploy:** Now every time you push to GitHub, Vercel automatically deploys! 🚀
